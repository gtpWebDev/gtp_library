Relative - "RELATIVE TO SELF" or "NUDGE" from where static would put it

Relative with no adjustments acts broadly like static

The other elements just assume its where it would have been with static positioning

Note it moves the element in such a way that it actually overflows the items around it

Absolute - removes object so that flow ignores its existence

The other elements in this case literally assume that the absolute element no longer exists

IMPORTANTLY, the left, right, top, bottom adjustments are based on the position element of the parent

If the parent is static, the adjustments are to the top left of the screen - unlikely this is useful

If the parent is relative (or absolute), the adjustments are to the top left of the parent - VERY USEFUL

position: absolute; left: 80px; top: 20px (parent position is relative in this case)
