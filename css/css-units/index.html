CSS Units

A piece of advice is to use rem for font-sizes and px for everything else

this allows users to get larger fonts when they zoom in

em are fiddly and difficult to maintain

Using calc(16px + 0.5vw) to give nicely responsive text size

1.5em

Units based on root/html font-size (seems to default to root if both): (Users can often be given control of the base font-size of a website)

1.5rem
Viewport-relative: vh and vw
25% of viewport
10% of viewport
Viewport-relative based on max/min of dimensions of vmin and vmax
25% of min viewport - wide and tall get small
25% of max viewport - wide and tall get big
Variations include swh,swh - assumes all browser menus etc are open, so screen size is minimum of possible Variations include lwh,lwh - assumes all browser menus etc are closed, so screen size is maximum of possible

25% of large viewport
